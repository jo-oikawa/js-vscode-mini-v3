Typography Guidelines

  • Use --font-ui for all interface text (buttons, labels, navigation)
  • Use --font-mono for code, terminal output, and technical values
  • Default UI size is 12-13px; code editor uses 13px
  • Panel titles use 11px uppercase with letter-spacing: 0.06em

Button Styling Rules

  • Default buttons use transparent background with --border stroke
  • Primary buttons use accent color border with subtle background tint
  • Icon buttons are 28×26px with centered content
  • All buttons have 6px border-radius and 0.15s hover transition
  • Hover state adds slight background lightening and stronger border
